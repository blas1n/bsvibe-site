## 이벤트 수집

BSupervisor는 AI 에이전트의 모든 행동을 이벤트로 수집합니다:

- API 호출 (요청/응답)
- 파일 시스템 접근
- 외부 서비스 통신
- 코드 실행 결과

목표: **p99 < 50ms** 수집 지연시간.

## 규칙 엔진

### 기본 규칙 (8개)

BSupervisor는 다음 위협을 자동 감지합니다:

1. SQL Injection 시도
2. Prompt Injection 시도
3. PII (개인정보) 유출
4. 과도한 리소스 사용
5. 허가되지 않은 외부 API 호출
6. 민감 파일 접근
7. 비정상적 토큰 사용량
8. 반복적 오류 패턴

### 커스텀 규칙

대시보드에서 조직에 맞는 규칙을 추가할 수 있습니다.

## 비용 추적

에이전트별, 모델별 LLM 비용을 추적하고 이상 탐지를 수행합니다.
