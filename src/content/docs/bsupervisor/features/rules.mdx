## 기본 규칙

BSupervisor는 8개의 기본 안전 규칙이 내장되어 있습니다. 각 규칙은 활성화/비활성화할 수 있습니다.

## 커스텀 규칙 생성

대시보드의 **Rules** 페이지에서 새 규칙을 추가합니다:

1. 규칙 이름과 설명 작성
2. 매칭 조건 설정 (glob 패턴)
3. 심각도 선택 (info / warning / critical)
4. 알림 채널 설정 (Telegram / Slack)

## 규칙 평가 순서

규칙은 우선순위 순으로 평가됩니다. 첫 번째 매칭 규칙의 액션이 실행됩니다.
