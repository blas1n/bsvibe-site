## 사용 가능한 플러그인

BSage는 18개의 플러그인을 제공합니다.

### Input 플러그인

| 플러그인 | 설명 |
|---------|------|
| Telegram | Telegram 봇으로 메시지 수신 |
| Slack | Slack 채널 모니터링 |
| Discord | Discord 봇 연동 |
| Email | IMAP 이메일 수신 |
| RSS | RSS/Atom 피드 구독 |

### Process 플러그인

| 플러그인 | 설명 |
|---------|------|
| Summarizer | LLM 기반 자동 요약 |
| Tagger | 자동 태그 부여 |
| DangerAnalyzer | 위험 요소 자동 탐지 |

### Output 플러그인

| 플러그인 | 설명 |
|---------|------|
| S3 | Amazon S3 파일 업로드 |
| Vault | Obsidian Vault 연동 |

## 플러그인 설정

대시보드의 **Plugin Manager** 페이지에서 각 플러그인을 활성화하고 credential을 설정합니다.
