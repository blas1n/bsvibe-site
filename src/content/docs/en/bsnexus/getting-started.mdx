BSNexus is an orchestration platform where AI agents collaborate to build software.

## What BSNexus Does

- **AI Architect**: Transforms your ideas into structured projects
- **Automatic task breakdown**: Automatically splits projects into work units
- **Parallel execution**: Multiple AI agents work simultaneously
- **Real-time Kanban**: Track task progress on a visual board

## Quick Start

### 1. Create an Account

Sign in with your BSVibe account at [nexus.bsvibe.dev](https://nexus.bsvibe.dev).

![BSNexus Dashboard](/images/screenshots/bsnexus-dashboard.png)

### 2. Create a Project

Click the **New Project** button on the dashboard and enter a project name.

### 3. Talk to the Architect

Describe what you want to build in natural language on the **Architect** page:

> "Build me a to-do app with a React frontend and a FastAPI backend."

The Architect analyzes your requirements and generates tasks.

### 4. Track Progress

Monitor task progress in real time on the Kanban board on the **Board** page.
