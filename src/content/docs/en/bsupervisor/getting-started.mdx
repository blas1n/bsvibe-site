BSupervisor is a safety layer that monitors and audits AI agent behavior in real time.

## What BSupervisor Does

- **Real-time rule evaluation**: 8 built-in safety rules (SQL injection, PII leak, etc.)
- **Custom rules**: Create audit rules tailored to your organization
- **Cost tracking**: Monitor LLM costs per agent and per model
- **Daily reports**: Automatically generated audit reports

## Quick Start

### 1. Create an Account

Sign in with your BSVibe account at [supervisor.bsvibe.dev](https://supervisor.bsvibe.dev).

![BSupervisor Dashboard](/images/screenshots/bsupervisor-dashboard.png)

### 2. Connect Your Agents

On the **Settings** page, connect the AI agent platforms you want to monitor:
- BSNexus (automatic integration within the BSVibe ecosystem)
- OpenAI / Anthropic (API key based)
- Custom agents (Webhook)

### 3. Review Rules

On the **Rules** page, 8 built-in safety rules are enabled by default. You can add new rules or disable existing ones as needed.

### 4. Monitor the Dashboard

View event counts, rule violations, and cost trends in real time on the **Dashboard**.
