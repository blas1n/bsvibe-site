## Analytics 대시보드

사이드바에서 **Analytics**를 클릭하면 전체 사용량과 비용을 한눈에 확인할 수 있습니다.

![Analytics 대시보드](/images/screenshots/bsgateway-usage.png)

## 주요 지표

| 지표 | 설명 |
|------|------|
| **Total Requests** | 선택 기간 내 전체 API 호출 수 |
| **Total Tokens** | 입력 + 출력 토큰 합계 |
| **Estimated Cost** | 모델별 단가 기반 추정 비용 (USD) |
| **Avg Latency** | 모델별 평균 응답 시간 (ms) |

## 기간 필터

우측 상단에서 분석 기간을 선택합니다:

| 필터 | 범위 |
|------|------|
| **Today** | 오늘 00:00부터 현재까지 |
| **Last 7 days** | 최근 7일 |
| **Last 30 days** | 최근 30일 |

## 차트

### Request Volume

시간대별 요청 수를 라인 차트로 표시합니다. 피크 시간대와 사용 패턴을 파악할 수 있습니다.

### Model Distribution

모델별 토큰 사용 비율을 시각화합니다. 어떤 모델이 가장 많이 사용되는지 한눈에 확인할 수 있습니다.

## 비용 최적화 인사이트

Analytics는 비용 절감 기회를 자동으로 식별합니다:

- **과다 라우팅 감지**: 간단한 요청이 고비용 모델로 라우팅된 케이스 표시
- **비활성 모델**: 사용 빈도가 낮은 모델 식별 (비활성화 검토)
- **피크 패턴**: 특정 시간대에 비용이 급증하는 패턴 감지

## Audit Log

사이드바에서 **Audit Log**를 클릭하면 모든 API 호출의 상세 기록을 조회할 수 있습니다:

![Audit Log](/images/screenshots/bsgateway-audit.png)

각 요청에 대해 다음 정보가 기록됩니다:
- 요청/응답 시간
- 사용된 모델
- 라우팅 경로 (어떤 단계에서 매칭되었는지)
- 입력/출력 토큰 수
- API 키
