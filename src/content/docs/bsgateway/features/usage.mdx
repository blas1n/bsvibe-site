## 대시보드

BSGateway 대시보드에서 실시간 사용량을 확인할 수 있습니다.

### 주요 지표

- **총 요청 수**: 기간 내 전체 API 호출 수
- **총 토큰**: 입력/출력 토큰 합계
- **추정 비용**: 모델별 단가 기반 비용 계산
- **평균 응답 시간**: 모델별 레이턴시

### 필터

- 기간: 오늘 / 7일 / 30일 / 커스텀
- API 키별
- 모델별
- 라우팅 유형별 (passthrough / alias / pattern / auto)

## 비용 최적화 인사이트

대시보드는 비용 절감 기회를 자동으로 식별합니다:

- 고비용 모델로 라우팅되었지만 간단한 요청이었던 케이스
- 사용 빈도가 낮은 모델 (비활성화 검토)
- 피크 시간대 패턴
